📖 App பயன்படுத்துவது எப்படி?
1
புதிய Client சேர்க்க
🔵 கீழே உள்ள + பட்டன் அழுத்தவும்
📝 பெயர், மொபைல், கடன் தொகை, வாரத் தவணை போடுங்கள்
📸 விரும்பினால் படம் எடுக்கலாம்
✅ "Client சேர்க்கவும்" அழுத்தவும்
2
கட்டணம் பெற்றால் என்ன செய்வது?
👆 Client பெயரை அழுத்தவும்
💚 கீழே "+ கட்டணம்" பட்டன் அழுத்தவும்
💰 தொகை போட்டு "சேர்" அழுத்தவும்
📲 தானாகவே WhatsApp message போகும்!
3
நிலுவை (Due) உள்ளவர்களை எப்படி தெரிந்துகொள்வது?
🔴 சிவப்பு badge = கட்டணம் தாமதம்
🟡 மஞ்சள் badge = 2 நாளில் கட்டணம் வர வேண்டும்
🟢 பச்சை badge = சரியான நேரத்தில் உள்ளனர்
📲 "Reminder" அழுத்தி WhatsApp அனுப்பவும்
4
வட்டி சேர்க்க வேண்டுமா?
👆 Client பெயரை அழுத்தவும்
🟠 கீழே "📈 வட்டி" பட்டன் அழுத்தவும்
💸 தொகை confirm செய்யவும்
📲 Client-க்கு message போகும்
5
தினமும் Backup எடுக்கவும்
☁️ மேலே "Backup" tab அழுத்தவும்
💬 "WhatsApp Backup" அழுத்தவும்
📲 உங்கள் WhatsApp-க்கே file வரும்
✅ Data எப்போதும் safe-ஆக இருக்கும்!
💡 முக்கிய குறிப்புகள்
✅ Settings-ல் உங்கள் WhatsApp நம்பர் போடுங்கள்
✅ தினமும் ஒரு முறை Backup எடுங்கள்
✅ Phone மாற்றும்போது Backup file transfer செய்யுங்கள்
✅ Internet இல்லாமலும் App வேலை செய்யும்
